# ✨ Next Level Form – Signup Page

A modern and stylish signup form with glassmorphism design and real-time validation using JavaScript.

## 🚀 Features

* Beautiful glassmorphism UI design
* Input field validation (Name, Email, Password)
* Real-time error messages
* Password confirmation check
* Success message on valid submission

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📦 Project Structure

* index.html (includes HTML, CSS, and JavaScript)

## 🎯 Functionality

The form validates user input before submission:

* Ensures all fields are filled
* Validates email format
* Checks password length (minimum 6 characters)
* Confirms matching passwords
##💡 Purpose

This project is created to practice frontend form validation and modern UI design techniques.
